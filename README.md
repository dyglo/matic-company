# Maticent Industries Limited - The African Heritage

Maticent Industries Limited is a premium IT solutions and hardware provider dedicated to delivering enterprise-grade technology to the African market. This web application serves as the digital front for Maticent, showcasing their extensive product catalog, solutions, and corporate journey.

## üöÄ Features

- **Dynamic Hero Experience**: Immersive video-based hero section with premium typography.
- **Interactive Showcases**: GSAP-powered pinned panels and animated text stories.
- **Product Catalog**: Comprehensive listing of laptops, desktops, monitors, and accessories.
- **Solution Sheets**: Detailed overview of IT solutions for various industries.
- **News & Insights**: Integrated blog/news section for company updates.
- **Modern UI/UX**: Built with a focus on premium aesthetics, featuring glassmorphism and smooth animations.
- **Data-Driven**: Powered by a robust data layer for products and news.

## üèÉ Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd project
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Set up environment variables:
   Create a `.env` file in the root directory and add your Supabase credentials.

5. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
